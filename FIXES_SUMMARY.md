## 🚀 Как Запустить

### Backend (Docker):

```bash
cd backend
start-docker.bat          # Windows
# или
bash start-docker.sh      # Linux/macOS
```

### Frontend:

```bash
cd frontend
flutter pub get
flutter run               # Mobile/Desktop
# или
flutter build web         # Web версия
```

### Доступ:
- **Backend API:** http://localhost:8000/docs
- **Frontend Web:** http://localhost:8080 (после `flutter build web` и部署)
